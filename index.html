Cache Memory

Cache Memory is a special very high-speed memory. The cache is a smaller and faster memory that stores copies of the data from frequently used main memory locations. There are various different independent caches in a CPU, which store instructions and data. The most important use of cache memory is that it is used to reduce the average time to access data from the main memory.

keyboard_double_arrow_right Characteristics of Cache Memory :


Cache memory is an extremely fast memory type that acts as a buffer between RAM and the CPU. Cache Memory holds frequently requested data and instructions so that they are immediately available to the CPU when needed. Cache memory is costlier than main memory or disk memory but more economical than CPU registers. Cache Memory is used to speed up and synchronize with a high-speed CPU.

keyboard_double_arrow_right Levels of Memory :


arrow_right Level 1 or Register:

It is a type of memory in which data is stored and accepted that are immediately stored in the CPU. The most commonly used register is Accumulator, Program counter, Address Register, etc.



arrow_right Level 2 or Cache memory:

It is the fastest memory that has faster access time where data is temporarily stored for faster access.



arrow_right Level 3 or Main Memory:

It is the memory on which the computer works currently. It is small in size and once power is off data no longer stays in this memory.



arrow_right Level 4 or Secondary Memory:

It is external memory that is not as fast as the main memory but data stays permanently in this memory.
